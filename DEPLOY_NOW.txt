STEP 2: PUSH YOUR CODE
-----------------------
After creating the repo, GitHub will show you commands.
Copy YOUR repository URL (looks like: https://github.com/YOUR-USERNAME/music-player.git)

Then run these commands in your terminal:

git remote add origin https://github.com/YOUR-USERNAME/music-player.git
git branch -M main
git push -u origin main

Option B: HEROKU
----------------
1. Install Heroku CLI: https://devcenter.heroku.com/articles/heroku-cli
2. Run: heroku login
3. Run: heroku create music-player-app
4. Run: git push heroku main
5. Run: heroku open
